Add Tabs

Vertical Tabs

To make a multi stepped layout, we offer a tabbed layout which can be used to create a tabular structure. From the story book example, we can notice wrapping a Form in a Panel with ":type" : "custom:vertical-tab" can create the following structure. From the documentation a two step form can be creating using the following JSON

{
  "items": [
    {
      ":type": "custom:vertical-tab",
      "name": "tabs",
      ...
      "items": [
        {
          "name": "textInput",
          ...
          "fieldType": "text-input"
        },
        {
          "name": "item3",
          ...
          "fieldType": "panel",
          "items": [
            {
              "name": "firstName",
              ...
              "fieldType": "text-input"
            }
          ]
        }
      ]
    }
  ]
}

Add tabs in our form

To add tabs in our login form as per the requirement we need to make modification to our json.

You can directly copy the contents of the JSON Document to see the tabs in your form here. Just ensure you keep the action property present in the original json downloaded from AEM

  1. Create a Panel Wizard as the first child of the Form element
{
    "name": "wizard",
    "label": {
        "value": "Welcome"
    },
    ":type": "custom:vertical-tab",
    "items" : []
}
  1. Inside the newly created Panel add two Panels : step1 and step2
{
    "name": "wizard",
    "label": {
        "value": "Welcome"
    },
    ":type": "custom:vertical-tab",
    "items" : [{
        "name" : "step1",
        "fieldType" : "panel",
        "label": {
            "value": "Identify Yourself"
        },
        "items": []
    },
    {
        "name" : "step2",
        "fieldType" : "panel",
        "label": {
            "value": "Verify OTP"
        },
        "items": []
    }
    ]
}
  1. In the items of step1 panel move the fields mobileNumber, creditCardEndingNumber, viewEligibilityButton. After this step the step1 panel would have the following json
{
        "name" : "step1",
        "fieldType" : "panel",
        "label": {
            "value": "Identify Yourself"
        },
        "items": [
        {
            "name": "mobileNumber",
            "fieldType": "text-input",
            "type": "string",
            "required": true,
            "description": "You do not need to add +91 - Share your 10 digit mobile number.\n We will be sending you an OTP to this number have it handy.",
            "label": {
                "value": "Please share your registered mobile number"
            }
        },
        {
            "name": "creditCardEndingNumber",
            "fieldType": "text-input",
            "type": "string",
            "required": true,
            "description": "We will need your credit card number to check your eligibility",
            "label": {
                "value": "Share last 4 digits of your credit card number"
            }
        },
        {
            "name": "viewEligibilityButton",
            "fieldType": "button",
            "label": {
                "value": "View Loan Eligibility 1"
            }
        }]
    }
  1. Similarly in the step2 panel add the fields identifyOtpValue and viewEligibilityConfirmButton.
  2. Add the remaining two panels from the form loanDetails and acountDetails as siblings of step1 and step2

After completing the steps, your file should look exactly like the JSON Document present here and look like the image below

Note that an extra action property would be present in the form downloaded from AEM
