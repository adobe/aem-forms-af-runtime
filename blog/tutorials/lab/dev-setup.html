Pre-requisites

For this lab, you will require knowledge to build application using react and the following software to set up a development environment.

  1. Java 11. Once installed ensure the java command works on the command line and JAVA_HOME environment variable is set in your system.
  2. Maven 3.5.X or later. Installation Instructions
  3. Node v16.X.X
  4. IDE - We will be using Visual Studio Code. If you are using Visual Studio Code, download the Adaptive Form Builder Extension as well. To install the extension follow the steps as per the official Microsoft documentation
  5. AEM Cloud SDK AEM SDK v2022.7.8005.20220711T194049Z-220600
  6. AEM Forms add-on v2022.07.06.02-220600

To download the supported version of Adobe Experience Manager as a Cloud Service SDK and AEM Forms add-on, Your Adobe Organization must be provisioned for AEM as a Cloud Service to download the AEM as a Cloud Service SDK

Before moving ahead ensure java, mvn, node, npm are configured in your path and are working on the command line.
