Authoring form json using VS Code Plugin

Download the Sample Form JSON for editing

  • Open your browser and enter the URL of the author instance. If you started the author instance with defaults as mentioned in the official docs, it should be http://localhost:4502/
  • Navigate to Adobe Experience Manager > Assets > Files > myheadlessform (The URL should be http://localhost:4502/assets.html/content/dam/myheadlessform)
  • Select the file af_model_sample.json and click on Download. A popup window will come and click Download again.
  • Save the file with the extension .form.json in the ui.frontend.react.forms.af module of the archetype project. To use the Adaptive Form Visual Studio Code Plugin, form json file needs to have a suffix of .form.json

Add a new Field in the Form

Using VS Code Plugin

  • If the extension is installed you will see the FormTree in the panel where you can see all the fields in the Form, and you can add a Field after any existing Field by right clicking on any field and select add.
  • Let's add a dateofbirth field after email field.
    • In the Form Tree, right click on the email field.
    • select add button and answer the prompts with the following answers
      • field type : date-input
      • field name : dateofbirth
      • field label: Date of Birth
    • A Field will be added in the Form Definition with the following content. You can update the properties like description, maximum, minimum or remove that altogether
{
      "name": "dateofbirth",
      "label": {
        "value": "Date of Birth"
      },
      "description": "",
      "type": "string",
      "required": false,
      "minimum": "2020-10-10",
      "maximum": "2021-10-10",
      "format": "date",
      "fieldType": "date-input"
    },

After update the JSON would look like this

    {
      "name": "dateofbirth",
      "fieldType": "date-input",
      "description": "Date of birth",
      "screenReaderText": "Enter your date of birth",
      "format": "date",
      "type": "string",
      "minimum": "2020-10-10",
      "maximum": "2021-10-10",
      "constraintMessages": {
        "minimum": "Please enter a date after 2020-10-10"
      },
      "label": {
        "value": "Date of Birth"
      }
    }

Without VS Code Plugin

  • Update the Form JSON in your desired text editor and add the following JSON snippet after the email field
    {
      "name": "dateofbirth",
      "fieldType": "date-input",
      "description": "Date of birth",
      "screenReaderText": "Enter your date of birth",
      "format": "date",
      "type": "string",
      "minimum": "2020-10-10",
      "maximum": "2021-10-10",
      "constraintMessages": {
        "minimum": "Please enter a date after 2020-10-10"
      },
      "label": {
        "value": "Date of Birth"
      }
    }
