Verify Submission

  • Open the contact form which was created earlier in form editor. Tap the Adaptive Forms Container component and Tap Properties. In the properties explorer, under submission select submit action as Submit to rest end point and check the enable post request checkbox. To input a value for URL for post request, go to https://requestbin.com/ and click on create a public bin, this would create a public HTTP end point in the following format (for example) https://enphg7yiuy00c.x.pipedream.net. Input this value in the URL for post request field and save the Properties dialog.

  • Navigate to the form on preview mode in AEM or on local development server, and on submit you should see the data on the public end point. API definition for submit end point can be checked here

If submission does not work, the step 2 mentioned here did not execute successfully
