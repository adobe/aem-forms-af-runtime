Start Local AEM Author Instance

Follow the instructions mentioned in the section Set up a local development environment and initial development project

  1. Download and extract the Adobe Experience Manager as a Cloud Service SDK
  2. Set up an Author instance. When launching the SDK Quickstart, include the argument -r prerelease. All headless specific features are behind pre-release (Publish instance is not necessary for this lab session)
  3. Access the local AEM Author instance, and access Help > “About Adobe Experience Manager” to validate the versions. Adobe Experience Manager version should be greater than or equal to: 2022.7.8005.xxx (with a suffix – PRERELEASE), and Adobe Experience Manager Forms version greater than or equal to: 2022.07.06.02.xxx.
